üìö How to Run the Permalist Project

  1. Install Node.js

    Download and install it from nodejs.org if you haven't already.
  2. Download the project

    Clone the repo or download and unzip the project folder.
  3. Open a terminal and navigate to the folder

    cd permalist
  4. Install dependencies
    Run: 
    npm install
    This will install:
    • bcrypt
    • body-parser
    • dotenv
    • ejs
    • express
    • express-session
    • pg
  5. Run the project
    Use: 
    npm start
    This will run the file index.js.

üóÑÔ∏è Set Up PostgreSQL Table

Run the following SQL in your PostgreSQL database (e.g., via pgAdmin):

Create Table: items

        CREATE TABLE items (
          id SERIAL PRIMARY KEY,
          title VARCHAR(100) NOT NULL,
          created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
          importance VARCHAR(20) DEFAULT 'Medium',
          user_id INTEGER NOT NULL,
          CONSTRAINT fk_user FOREIGN KEY (user_id) REFERENCES users(id),
          category VARCHAR(100)
        );

Create Table: users

        CREATE TABLE users (
          id SERIAL PRIMARY KEY,
          username VARCHAR(255) NOT NULL UNIQUE,
          password VARCHAR(255)
        );

Create Table: deleted_items

        CREATE TABLE deleted_items (
          user_id INTEGER PRIMARY KEY REFERENCES users(id),
          deleted_count INTEGER DEFAULT 0
        );

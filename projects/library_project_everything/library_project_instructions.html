üìö How to Run the Library Project

  1. Install Node.js
    Download and install it from nodejs.org if you haven't already.
  2. Download the project
    Clone the repo or download and unzip the project folder.
  3. Open a terminal and navigate to the folder 
    cd library_project
  4. Install dependencies
    Run: 
    npm install
    This will install:
    • axios
    • body-parser
    • dotenv
    • ejs
    • express
    • pg
  5. Run the project
    Use: 
    npm start
    This will run the file server.js.

üóÑÔ∏è Set Up PostgreSQL Table

Run the following SQL in your PostgreSQL database (e.g., via pgAdmin):

Create Table: books

      CREATE TABLE books (
        id SERIAL PRIMARY KEY,
        title VARCHAR(100),
        recomandation INT,
        summary TEXT,
        isbn VARCHAR(17)
      );

Create Table: categories

      CREATE TABLE  categories (
        id SERIAL PRIMARY KEY ,
        name_of_category TEXT,
        book_id INTEGER,
        FOREIGN KEY (book_id) REFERENCES books(id)
      );

Create Table: my_opinion

      CREATE TABLE my_opinion (
        id SERIAL PRIMARY KEY ,
        personal_opinion TEXT,
        book_id INTEGER,
        FOREIGN KEY (book_id) REFERENCES books(id)
      );

Create Table: quotes

      CREATE TABLE quotes (
        id SERIAL PRIMARY KEY ,
        quotes TEXT,
        quotes_id INTEGER,
        FOREIGN KEY (quotes_id) REFERENCES books(id)
      );
